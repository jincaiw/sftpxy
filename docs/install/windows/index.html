English

Recommended: EXE installer

  1. Download SFTPxy_v0.2.2_windows_x86_64.exe from Downloads.
  2. Right-click the installer and choose Run as administrator.
  3. Finish the wizard. The Windows service is installed and started automatically.
  4. Open WebAdmin: http://localhost:30080/web/admin/login.

Installed locations

  • Program: C:\Program Files\SFTPxy
  • Data and config: C:\ProgramData\SFTPxy
  • Main config: C:\ProgramData\SFTPxy\SFTPxy.json
  • Env overrides: C:\ProgramData\SFTPxy\env.d

Portable ZIP

  1. Download SFTPxy_v0.2.2_windows_portable.zip.
  2. Extract it to a folder.
  3. Run PowerShell in that folder:
.\SFTPxy.exe serve -c .

中文

推荐：EXE 安装程序

  1. 从 下载页 获取 SFTPxy_v0.2.2_windows_x86_64.exe。
  2. 右键安装程序，选择 以管理员身份运行。
  3. 按向导完成安装。Windows 服务会自动安装并启动。
  4. 打开 WebAdmin：http://localhost:30080/web/admin/login。

安装位置

  • 程序目录：C:\Program Files\SFTPxy
  • 数据和配置：C:\ProgramData\SFTPxy
  • 主配置：C:\ProgramData\SFTPxy\SFTPxy.json
  • 环境变量覆盖：C:\ProgramData\SFTPxy\env.d

便携 ZIP

  1. 下载 SFTPxy_v0.2.2_windows_portable.zip。
  2. 解压到一个目录。
  3. 在该目录打开 PowerShell：
.\SFTPxy.exe serve -c .
