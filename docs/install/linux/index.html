English

Use DEB/RPM packages when possible. Use the portable archive only when you do not want system packages.

DEB package: Debian / Ubuntu

  1. Download SFTPxy_0.2.2-1_amd64.deb from Downloads.
  2. Install it:
sudo apt install ./SFTPxy_0.2.2-1_amd64.deb
sudo systemctl enable --now SFTPxy
sudo systemctl status SFTPxy

RPM package: RHEL / Fedora / Rocky / openSUSE

  1. Download SFTPxy-0.2.2-1.x86_64.rpm.
  2. Install it:
sudo dnf install ./SFTPxy-0.2.2-1.x86_64.rpm
sudo systemctl enable --now SFTPxy
sudo systemctl status SFTPxy

If your distro uses yum:

sudo yum install ./SFTPxy-0.2.2-1.x86_64.rpm

Portable archive

tar -xf SFTPxy_v0.2.2_linux_x86_64.tar.xz
cd SFTPxy_v0.2.2_linux_x86_64
./SFTPxy serve -c .

First login

Open http://localhost:30080/. If this is your first local run and no admin exists, start once with bootstrap variables:

SFTPXY_DATA_PROVIDER__CREATE_DEFAULT_ADMIN=1 \
SFTPXY_DEFAULT_ADMIN_USERNAME=admin \
SFTPXY_DEFAULT_ADMIN_PASSWORD='change-this-password' \
SFTPXY_COMMON__SECRET_MIN_ENTROPY=0 \
./SFTPxy serve -c .

中文

推荐优先使用 DEB/RPM 安装包；不想安装到系统时再使用便携压缩包。

DEB 包：Debian / Ubuntu

  1. 从 下载页 获取 SFTPxy_0.2.2-1_amd64.deb。
  2. 执行安装：
sudo apt install ./SFTPxy_0.2.2-1_amd64.deb
sudo systemctl enable --now SFTPxy
sudo systemctl status SFTPxy

RPM 包：RHEL / Fedora / Rocky / openSUSE

  1. 下载 SFTPxy-0.2.2-1.x86_64.rpm。
  2. 执行安装：
sudo dnf install ./SFTPxy-0.2.2-1.x86_64.rpm
sudo systemctl enable --now SFTPxy
sudo systemctl status SFTPxy

如果系统使用 yum：

sudo yum install ./SFTPxy-0.2.2-1.x86_64.rpm

便携压缩包

tar -xf SFTPxy_v0.2.2_linux_x86_64.tar.xz
cd SFTPxy_v0.2.2_linux_x86_64
./SFTPxy serve -c .

首次登录

打开 http://localhost:30080/。如果首次本地运行且还没有管理员，可以临时使用下面变量启动：

SFTPXY_DATA_PROVIDER__CREATE_DEFAULT_ADMIN=1 \
SFTPXY_DEFAULT_ADMIN_USERNAME=admin \
SFTPXY_DEFAULT_ADMIN_PASSWORD='change-this-password' \
SFTPXY_COMMON__SECRET_MIN_ENTROPY=0 \
./SFTPxy serve -c .
