English

SFTPxy is distributed as a portable archive for macOS.

Portable run

  1. Download the macOS archive that matches your CPU.
  2. Extract the archive.
  3. Start SFTPxy.
./SFTPxy serve -c .

The repo includes init/com.github.jincaiw.SFTPxy.plist. It expects the Homebrew-style layout under /usr/local/opt/SFTPxy.

  • Config: /usr/local/etc/SFTPxy
  • Log file: /usr/local/opt/SFTPxy/var/log/SFTPxy.log

中文

macOS 版本以便携压缩包形式发布。

便携运行

  1. 下载与你的 CPU 匹配的 macOS 压缩包。
  2. 解压文件。
  3. 启动 SFTPxy。
./SFTPxy serve -c .

仓库内提供 init/com.github.jincaiw.SFTPxy.plist。它默认适配 Homebrew 风格目录 /usr/local/opt/SFTPxy。

  • 配置：/usr/local/etc/SFTPxy
  • 日志：/usr/local/opt/SFTPxy/var/log/SFTPxy.log
