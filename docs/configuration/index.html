FTP passive range30085-30088

Config search path

  • Linux: $HOME/.config/SFTPxy, /etc/SFTPxy, /usr/local/etc/SFTPxy
  • macOS: /usr/local/etc/SFTPxy
  • Windows: C:\ProgramData\SFTPxy

Best practice

  • Put secrets in environment variables.
  • Keep platform-specific overrides outside the main config file.
  • Back up config, data, and logs separately.
